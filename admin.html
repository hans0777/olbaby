📅 Google Calendar 연동

예약 등록 시 자동으로 Google 캘린더에 일정이 추가됩니다

미연동

Google Cloud Console에서 발급받은 클라이언트 ID

Google Cloud Console에서 발급받은 API Key

🔧 설정 방법

  1. Google Cloud Console에서 새 프로젝트 생성
  2. "API 및 서비스" → "라이브러리"에서 "Google Calendar API" 활성화
  3. "사용자 인증 정보" → "OAuth 2.0 클라이언트 ID" 생성 (웹 애플리케이션)
  4. "API 키" 생성
  5. 위에 클라이언트 ID와 API Key 입력 후 "연동하기" 클릭
