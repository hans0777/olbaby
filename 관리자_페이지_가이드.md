# 오엘베이비 고객관리 시스템 - 관리자 페이지 가이드

## 📸 시스템 개요

오엘베이비 고객관리 시스템은 예약부터 사후 마케팅까지 전체 고객 여정을 자동화하는 통합 관리 솔루션입니다.

### 핵심 기능
1. **예약 시 자동 일정 등록** - Google Calendar 자동 동기화
2. **D-1 자동 알림** - 촬영 1일 전 자동 문자 발송
3. **촬영 당일 혜택 안내** - 당일 한정 구매 혜택 문자
4. **D+1 설문 요청** - 촬영 후 설문 및 혜택 지급
5. **2개월마다 할인권 발송** - 기존 고객 재방문 유도

---

## 🚀 빠른 시작

### 1단계: 로그인
- **비밀번호**: `olbaby2026` (admin.html 파일에서 변경 가능)
- 로그인 후 대시보드가 표시됩니다.

### 2단계: Mock 데이터 모드
- Supabase 설정 없이도 즉시 체험 가능
- 샘플 고객, 예약, 설문 데이터가 자동으로 로드됩니다.

## 📊 대시보드 구성

### 통계 카드
- **전체 고객**: 등록된 전체 고객 수
- **전체 예약**: 등록된 전체 예약 건수
- **오늘 촬영**: 오늘 예정된 촬영 건수
- **예약확정**: 확정된 예약 건수
- **설문응답**: 수집된 설문 응답 수

## 🗓️ 1. 일정 캘린더

### 기능
- **월별 캘린더 뷰**: 예약 일정을 한눈에 확인
- **상태별 색상 구분**
  - 🔵 파란색: 예약확정
  - 🟢 초록색: 촬영완료
  - 🟡 노란색: 상담중
  - ⚪ 회색: 취소
- **날짜 클릭**: 해당 날짜의 상세 일정 표시

### 사용법
1. 캘린더에서 날짜를 클릭
2. 해당 날짜의 예약 목록 확인
3. 문자 발송 상태 확인 (✅ 발송완료, ☐ 미발송)

---

## 👥 2. 고객 목록

### 고객 등록
1. **"+ 고객 추가"** 버튼 클릭
2. 필수 정보 입력:
   - 고객명 (필수)
   - 연락처 (필수, 형식: 010-1234-5678)
   - 아기 이름
   - 아기 생년월일
   - 상담 경로 (카톡, 전화, 인스타DM, 네이버톡톡)
   - 유입 경로 (인스타그램, 블로그, 네이버검색, 지인추천, 맘카페)
   - 메모
3. **"고객 등록"** 클릭
4. ✅ 등록 완료 후 바로 예약 등록 가능

### 고객 관리
- **검색**: 이름, 연락처, 고객ID로 검색
- **상세**: 고객 정보 상세 보기
- **수정**: 고객 정보 수정
- **삭제**: 고객 삭제 (관련 예약도 함께 삭제)

---

## 📅 3. 예약 현황

### 예약 등록
1. **"+ 예약 추가"** 버튼 클릭
2. 고객 선택:
   - 검색창에서 이름/연락처 검색
   - 목록에서 선택
   - 또는 **"+ 신규 고객"** 버튼으로 바로 고객 등록
3. 예약 정보 입력:
   - 촬영 날짜 (필수)
   - 촬영 시간 (필수)
   - 촬영 상품 (필수)
   - 예약 상태 (상담중/예약확정/촬영완료/취소)
   - 메모
4. **"예약 등록"** 클릭
5. ✅ Google Calendar에 자동 등록 (설정 시)

### 문자 발송 관리
각 예약별로 4단계 문자 발송 상태 표시:
- **📨 확정**: 예약 확정 문자
- **📨 D-1**: 촬영 1일 전 안내
- **📨 당일**: 당일 혜택 안내
- **📨 설문**: 설문 요청

#### 수동 발송
1. 발송 버튼(📨) 클릭
2. 문자 내용 확인/수정
3. **"발송하기"** 클릭
4. ✅ 발송 완료 (알리고 API 연동 시)

---

## 💬 4. 문자 발송

### 수동 문자 발송
1. 수신자 연락처 입력
2. 문자 템플릿 선택 (또는 직접 입력):
   - 예약확정
   - D-1안내
   - 당일혜택
   - 설문요청
3. 내용 확인/수정
4. **"문자 발송"** 클릭

### 문자 템플릿
미리 작성된 템플릿 사용 가능:
- 예약확정: 촬영 일정, 준비물 안내
- D-1안내: 촬영 전 가이드 링크
- 당일혜택: 액자/앨범 할인 안내
- 설문요청: 설문 참여 링크 + 혜택 안내

---

## 📝 5. 설문 응답

### 기능
- **설문 응답 조회**: 고객별 설문 결과 확인
- **만족도**: ⭐ 별점 (1-5)
- **추천의향**: NPS 점수 (0-10)
- **혜택지급**: 지급완료/미지급 상태 관리

### 설문 분석
- 좋았던 점 확인
- 개선사항 확인
- 유입경로 분석

---

## 💌 6. 할인권 관리 (NEW!)

### 통계
- **총 발송 횟수**: 전체 할인권 발송 건수
- **이번 달 발송**: 당월 발송 건수
- **다음 발송 대상**: 발송 대기 고객 수

### 자동 발송 대상
- 촬영 완료 후 60일(2개월) 경과 고객
- 마지막 할인권 발송 후 60일 경과 고객

### 사용법

#### 개별 발송
1. **"📬 다음 발송 대기 고객"** 섹션 확인
2. 고객별 **"할인권 발송"** 버튼 클릭
3. 발송 확인

#### 일괄 발송
1. **"💌 2개월 경과 고객에게 발송"** 버튼 클릭
2. 발송 대상 확인
3. 일괄 발송 실행

### 할인권 내용 커스터마이징
- 설정 탭 > 자동 마케팅 설정에서 수정 가능
- 예: "재방문 고객 특별 할인 20%"

## ⚙️ 7. 시스템 설정

### 🤖 자동 마케팅 설정

#### 1단계: 예약 시 자동 일정 등록
- ✅ 활성화 시: Google Calendar 자동 동기화
- 예약 등록 시 자동으로 캘린더 이벤트 생성

#### 2단계: D-1 자동 알림
- ✅ 활성화 시: 매일 오전 9시 자동 발송
- 내일 촬영 예정 고객에게 안내 문자 전송

#### 3단계: 촬영 당일 혜택 안내
- ✅ 활성화 시: 촬영 시작 1시간 후 자동 발송
- 발송 시점 선택 가능 (즉시/1시간 후/2시간 후)

#### 4단계: D+1 설문 요청
- ✅ 활성화 시: 매일 오후 3시 자동 발송
- 전날 촬영한 고객에게 설문 요청 전송

#### 5단계: 2개월마다 할인권 발송
- ✅ 활성화 시: 매월 1일 오전 10시 자동 발송
- 발송 주기 설정 (30일/60일/90일)
- 할인 내용 커스터마이징

#### 자동화 실행 시간
- **D-1 알림**: 매일 오전 9시
- **설문 요청**: 매일 오후 3시
- **당일 혜택**: 촬영 완료 시 즉시
- **할인권 발송**: 매월 1일 오전 10시

### 📅 Google Calendar 연동

#### 설정 방법
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. 새 프로젝트 생성
3. "API 및 서비스" → "라이브러리" → "Google Calendar API" 활성화
4. "사용자 인증 정보" → "OAuth 2.0 클라이언트 ID" 생성
5. "API 키" 생성
6. admin.html 설정 탭에 정보 입력
7. **"Google Calendar 연동하기"** 클릭
8. Google 계정 인증
9. ✅ 연동 완료

#### 기능
- 예약 자동 등록
- 1일 전, 1시간 전 자동 알림 설정
- 고객에게 캘린더 초대 이메일 발송 (선택)

### 📱 알리고 SMS 연동

#### 설정 방법
1. [알리고 SMS](https://smartsms.aligo.in/) 회원가입
2. 발신번호 등록 (본인 인증 필요)
3. API Key 발급받기
4. admin.html 설정 탭에 정보 입력:
   - API Key
   - User ID
   - 발신번호
5. **"저장 및 테스트"** 클릭
6. ✅ 테스트 문자 발송 확인

#### 요금
- SMS (90자 이하): 건당 약 15원
- LMS (90자 초과): 건당 약 50원
- 자동으로 문자 길이에 따라 SMS/LMS 선택

### 💬 카카오톡 알림톡 연동

#### 주의사항
- 카카오톡 채널 개설 필요
- 알림톡 템플릿 사전 등록 및 승인 필요 (1-2일 소요)
- 비즈니스 인증 필요

#### 권장 사항
- **알리고 SMS를 먼저 사용하는 것을 권장**
- 알리고에서 SMS + 알림톡 함께 사용 가능

## 🔧 문제 해결

### 문자 발송이 안 돼요
1. 알리고 SMS 설정 확인
2. API Key, User ID, 발신번호 재확인
3. 알리고 계정 잔액 확인
4. 발신번호 등록 상태 확인

### Google Calendar 연동이 안 돼요
1. API Key와 Client ID 재확인
2. Google Cloud Console에서 API 활성화 확인
3. OAuth 동의 화면 설정 확인
4. 브라우저 쿠키/캐시 삭제 후 재시도

### 자동화가 작동하지 않아요
1. 설정 탭에서 자동화 활성화 여부 확인
2. 브라우저 항상 실행 (자동화는 백엔드 필요)
3. **권장**: cron job 또는 서버리스 함수로 자동화 실행

### Mock 데이터에서 실제 데이터로 전환하려면?
1. Supabase 프로젝트 생성
2. 데이터베이스 테이블 생성 (시작_가이드.md 참고)
3. admin.html 파일에서 SUPABASE_URL, SUPABASE_ANON_KEY 설정
4. 페이지 새로고침
