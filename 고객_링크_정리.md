## 🚀 배포 방법

### 1. GitHub Pages 배포 (무료)

```bash
# 1. GitHub 저장소 생성
git init
git add *.html
git commit -m "Add customer pages"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/olbaby.git
git push -u origin main

# 2. GitHub 저장소 Settings > Pages
# - Source: Deploy from a branch
# - Branch: main
# - 저장

# 3. 배포 완료 후 URL
https://YOUR_USERNAME.github.io/olbaby/
```

### 2. 커스텀 도메인 설정 (선택)

```
예시: olbaby.com
- guide.html → https://olbaby.com/guide
- offer.html → https://olbaby.com/offer
- survey.html → https://olbaby.com/survey
```

## 💡 URL 단축 서비스 활용

문자 메시지는 글자 수 제한이 있으므로 URL 단축을 권장합니다.

### 추천 서비스
1. **비틀리 (Bitly)** - https://bitly.com
   ```
   원본: https://your-username.github.io/olbaby/guide.html
   단축: https://bit.ly/olbaby-guide
   ```

2. **네이버 단축 URL** - https://me2.do
   ```
   원본: https://your-username.github.io/olbaby/offer.html
   단축: https://me2.do/olbaby-offer
   ```

3. **구글 URL 단축** (Firebase Dynamic Links)
   ```
   원본: https://your-username.github.io/olbaby/survey.html
   단축: https://olbaby.page.link/survey
   ```
