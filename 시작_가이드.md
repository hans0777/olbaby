## 📋 즉시 운영 4단계

### 1단계: Supabase 설정 (5분)

#### 1-1. 프로젝트 생성
1. https://supabase.com 접속 → 회원가입/로그인
2. "New Project" 클릭
   - **Project Name**: `olbaby-crm`
   - **Database Password**: 안전한 비밀번호 설정 (메모 필수!)
   - **Region**: `Northeast Asia (Seoul)`
3. "Create new project" 클릭 → 약 2분 대기

#### 1-2. API 키 확인
1. 프로젝트 생성 완료 후 좌측 **Settings** → **API** 클릭
2. 아래 두 값 복사:
   ```
   Project URL: https://xxxxxxxxx.supabase.co
   anon public key: eyJhbGci...
   ```

#### 1-3. 데이터베이스 생성
1. 좌측 **SQL Editor** 클릭
2. "New query" 클릭
3. `supabase_schema.sql` 파일 내용 전체 복사 → 붙여넣기
4. **Run** 버튼 클릭 (⏯️) → "Success" 확인
5. 좌측 **Table Editor**에서 테이블 5개 확인:
   - ✅ customers (고객DB)
   - ✅ reservations (예약관리)
   - ✅ sms_templates (문자템플릿)
   - ✅ survey_responses (설문응답)
   - ✅ events (이벤트관리)

## 🔧 문제 해결

### 설문 제출이 안 될 때
1. `survey.html`의 Supabase URL/Key 확인
2. Supabase → Table Editor → `survey_responses` 테이블 존재 확인
3. 브라우저 콘솔(F12) → Console 탭에서 에러 메시지 확인

### 관리자 페이지 데이터가 안 보일 때
1. `admin.html`의 Supabase URL/Key 확인
2. 브라우저 콘솔(F12)에서 네트워크 요청 확인
3. Supabase → Table Editor에서 수동으로 데이터 추가해보기

### 문자 발송이 안 될 때
1. 알리고 API 키 확인
2. 알리고 포인트 잔액 확인
3. 발신번호 등록 여부 확인
